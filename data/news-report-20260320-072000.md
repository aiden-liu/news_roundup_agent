- **Published**: March 19, 2026

NVIDIA has introduced KV Cache Transform Coding (KVTC), a groundbreaking technique that compresses the key-value cache in large language models by up to 20x without requiring any model weight modifications. This innovation applies concepts from media compression formats like JPEG to AI systems, dramatically reducing GPU memory requirements and cutting time-to-first-token by up to 8x for multi-turn conversational applications. The technique works by identifying redundant tokens in the attention mechanism's cache and compressing them while preserving semantic information. For developers working with smaller models or edge devices where memory is constrained, this could be a game-changer. The method is particularly valuable for coding assistants and agentic AI workloads that require rapid response times.

- **URL**: https://www.theregister.com/2026/03/16/nvidia_lpx_groq_3
- **Published**: March 16, 2026

At GTC 2026, NVIDIA announced the integration of Groq's Language Processing Units (LPUs) into its Vera Rubin platform for the first time. The Groq 3 LPX rack system contains 256 interconnected LPUs connected via a custom Spectrum-X interconnect, delivering deterministic inference performance with 40 petabytes per second of bandwidth. This marks NVIDIA's first dedicated non-GPU inference hardware, addressing the growing demand for low-latency AI workloads that GPUs cannot efficiently serve. The LPX architecture is designed specifically for multi-agent systems and real-time applications where consistent response times are critical.
