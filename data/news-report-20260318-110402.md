The LPX racks contain 256 Groq 3 LPUs with 128GB of on-chip SRAM and 40 petabytes per second of bandwidth, connected via a custom Spectrum-X interconnect to neighboring Vera-Rubin NVL72 training racks. This hybrid architecture targets multi-agent AI systems requiring ultra-low latency inference, with the Groq 3 LPU capable of processing 1,500 tokens/second for agentic workloads. The move represents a strategic expansion beyond GPU-centric infrastructure, addressing the growing demand for specialized inference hardware in the era of complex AI agent ecosystems.

### Nvidia says it can shrink LLM memory 20x without changing model weights | VentureBeat

- **Source**: VentureBeat
- **URL**: https://venturebeat.com/orchestration/nvidia-shrinks-llm-memory-20x-without-changing-model-weights
- **Published**: 2026-03-17

Nvidia has unveiled two groundbreaking memory optimization techniques: KV Cache Transform Coding (KVTC) and Dynamic Memory Sparsification (DMS). KVTC compresses LLM key-value cache by 20x without requiring model retraining, while DMS reduces reasoning costs by 8x through intelligent token eviction strategies. These innovations enable smaller models to achieve enterprise-grade performance at a fraction of the memory footprint. For multi-turn AI applications and coding assistants, these techniques cut time-to-first-token by up to 8x while maintaining accuracy. The research demonstrates that inference-time optimization can rival pre-training efficiency gains, fundamentally changing how organizations deploy and scale LLMs in production environments.
