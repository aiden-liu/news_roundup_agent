### Open SWE: An Open-Source Framework for Internal Coding Agents

- **Source**: LangChain
- **URL**: https://blog.langchain.com/open-swe-an-open-source-framework-for-internal-coding-agents/
- **Published**: March 17, 2026

LangChain has released Open SWE, an open-source framework that captures the architectural patterns used by successful internal coding agents at major tech companies including Stripe (Minions), Ramp (Inspect), and Coinbase (Cloudbot). Built on Deep Agents and LangGraph, Open SWE provides core components for building autonomous coding agents with a four-agent architecture: Manager, Planner, Programmer, and Reviewer. The Reviewer component checks outputs for errors before any pull request is created, aiming to reduce broken builds and review cycles. The entire project is open source, designed to be extended and customized for different codebases and workflows.
