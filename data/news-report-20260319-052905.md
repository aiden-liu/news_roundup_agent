### Nvidia's KVTC Compresses LLM Memory 20x Without Model Changes | VentureBeat

- **Source**: VentureBeat
- **URL**: https://venturebeat.com/orchestration/nvidia-shrinks-llm-memory-20x-without-changing-model-weights
- **Published**: March 17, 2026

Nvidia has unveiled a groundbreaking technique called **KV Cache Transform Coding (KVTC)** that compresses the key-value cache in large language models by up to 20x without requiring any changes to the model weights or architecture. This innovation applies concepts from media compression formats like JPEG to AI systems, significantly reducing GPU memory demands and cutting time-to-first-token by up to 8x for multi-turn AI applications. The technique is particularly impactful for coding assistants and agentic AI workloads, where maintaining context over extended conversations becomes increasingly expensive.

### Nvidia Groq 3 LPU Joins Vera Rubin Platform at GTC | The Register

- **Source**: The Register
- **URL**: https://theregister.com/2026/03/16/nvidia_lpx_groq_3
- **Published**: March 16, 2026

At GTC 2026, Nvidia announced the integration of Groq's Language Processing Unit (LPU) into its Vera Rubin platform for the first time. The new **Groq 3 LPU** is designed specifically for low-latency inference in multi-agent systems, featuring 128GB of on-chip SRAM and 40 petabytes per second of bandwidth. Nvidia plans to deploy these chips in dedicated LPX racks containing 256 interconnected LPUs, connected via a custom Spectrum-X interconnect to neighboring Vera-Rubin NVL72 training racks. This marks Nvidia's first non-GPU inference accelerator, acknowledging that GPUs alone cannot serve the rapidly growing inference market efficiently.

## Key Trends

- **Inference Optimization Over Training**: The industry is pivoting from scaling compute for model training to optimizing memory and latency for inference. Techniques like KVTC are enabling smaller models to run efficiently on existing hardware.

- **Specialized Inference Hardware Emerges**: Groq's LPU represents a paradigm shift, offering dedicated inference acceleration that complements rather than competes with GPUs. This hybrid approach may become standard in data center deployments.

- **Memory Compression as Enabler**: Novel compression techniques are making it possible to run larger models on smaller hardware, democratizing access to advanced AI capabilities.

- **Multi-Agent Systems Drive Demand**: The rise of agentic AI workloads is creating unique requirements for low-latency, high-throughput inference that traditional GPUs struggle to meet efficiently.

- **Ecosystem Fragmentation**: While Nvidia maintains dominance in training infrastructure, the inference layer is becoming more fragmented with specialized hardware from multiple vendors.
